3. Orientado a evento - I/O não bloqueante

Exemplo de código bloqueante:

data = downloadURL('http://some.api/people/1')
person = JSON.decode(data)
print(person.name);

Exemplo de código não bloqueante:

http.get('http://some.api/people/1', function(err, res) {
	var person = JSON.parse(res.data)
	console.log(person.name)
});

Exemplo com jQuery:

$.get('http://some.api/people/1', funciton(data) {
	console.log(data.person.name)
});

6. Inúmeras ferramentas

 

Instale qualquer biblioteca com apenas uma linha de código:

$ npm install nome-do-pacote

Defina as dependências do seu projeto com o package.json:

{
  "name": "nome-projeto",
  "version": "1.0.0",
  "dependencies": {
    "underscore": "~1.6.0",
    "express": "~4.9.0",
    "express-load": "~1.1.14",
    "sequelize": "~2.0.0-rc2"
  }
}

E use o npm para instalar tudo de uma vez só:

$ npm install
